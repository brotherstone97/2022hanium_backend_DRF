# Environment
+ Python 3.9.7
+ Django 4.0.4
+ Django REST Framework 3.13.1
+ PyMongo 3.12.3(4.1.1로 변경 예정)

# 주의사항
+ venv생성: ```python3 -m venv /path/to/new/virtual/environment```
+ venv실행: ```source venv(가상환경 디렉토리)/bin/activate```
+ venv종료: ```deactivate```

# Deployment
```python3 manage.py runserver 0.0.0.0:8000```
